# Educational CRM System

[![GitHub license](https://img.shields.io/badge/license-MIT-blue.svg)](https://github.com/Shubham9599/Educational-CRM-System)

A **full-stack Education CRM system** to manage student data, course enrollments, and academic progress. Built using **Java, Spring Boot, MySQL, and Thymeleaf**, this system provides a responsive interface for administrators and students to efficiently handle academic operations.

---

## Features

- Student registration and management
- Course assignment and tracking
- Admin dashboard for managing students, courses, and reports
- Responsive UI built with **HTML, CSS, Bootstrap, and Thymeleaf**
- CRUD operations for students, courses, and assignments
- Secure data storage using **MySQL database**
- Optimized backend for faster query handling

---

## Tech Stack

- **Backend:** Java, Spring Boot, Spring MVC  
- **Frontend:** HTML, CSS, Bootstrap, Thymeleaf  
- **Database:** MySQL  
- **Tools:** Maven, Git, Postman
